# TP3_ludique_vds
tp2 programation ludique, Vasile Dominique Sulici.

## - Jeux platformer simple, avec systeme de clé et de progression.

# - HOW TO PLAY

-            Récupérer toutes les capsules pour pouvoir passer a travers le portail et passer au prochain niveau.

# - CONTROLS
-            W - SAUTER
-            A - GAUCHE
-            D - DROITE
